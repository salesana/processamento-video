Laboratório 02 - Filtragem

I. Introdução

O laboratório realizado consistiu em uma experiência introdutória prática de captura, edição e processamento de imagens e vídeos, via aplicações desenvolvidas com linguagem C++ e compilação através do software "OpenCV" instaciado em sistema operacional Ubuntu (Linux).

II. Fundamentos Básicos

A qualidade da captura de imagem e vídeo depende de diversos fatores, como a iluminação, resolução e configuração da câmera. A utilização de luz direta e difusa, juntamente com a correção de temperatura de cor, é crucial para obter resultados nítidos e bem iluminados. Durante o experimento, foram testadas diversas disposições de câmera e iluminação a fim de obtermos o melhor resultado. Além disso, a escolha da resolução afeta diretamente a clareza e o tamanho do arquivo final.

II. a) Objetivo

O objetivo deste experimento foi aplicar diversos filtros de suavização em uma imagem para observar os efeitos de diferentes técnicas de suavização e tamanhos de kernel sobre a qualidade da imagem. Foram utilizados os filtros Homogeneous (média), Gaussian (gaussiano), Median (mediana) e Bilateral, com kernels de tamanhos variados: 3x3, 5x5, 7x7 e 11x11. A análise buscou identificar o impacto de cada filtro e do tamanho do kernel na preservação de detalhes e remoção de ruídos.

    - Preparação da Imagem:

    Carregamos uma imagem de entrada (foto_grupo_1.png) utilizando a biblioteca OpenCV, que foi usada como base para a aplicação dos filtros de suavização.

    O procedimento experimental foi realizado em etapas, aplicando diferentes filtros de suavização em uma imagem base com o uso da biblioteca OpenCV. O objetivo era comparar os efeitos de cada filtro e observar como a variação do tamanho do kernel impacta a qualidade da imagem em termos de suavização e preservação de detalhes.

    - Aplicação dos Filtros de Suavização:

    a) Homogeneous Blur (Média): Este filtro calcula a média dos pixels vizinhos para suavizar a imagem. Foi implementado utilizando o método blur() da OpenCV.

    b) Gaussian Blur (Gaussiano): Utiliza uma distribuição gaussiana para suavizar a imagem, preservando os detalhes de forma mais eficaz. Implementado com GaussianBlur().

    c) Median Blur (Mediana): Substitui cada pixel pela mediana de seus vizinhos, sendo mais eficaz para remover ruídos pontuais. Implementado com medianBlur().

    d) Bilateral Blur: Aplica suavização enquanto preserva as bordas da imagem, sendo ideal para imagens que precisam manter detalhes nas bordas. Implementado com bilateralFilter().

    - Variação dos Tamanhos de Kernel:

    Para cada filtro, foram testados quatro tamanhos de kernel: 3x3, 5x5, 7x7 e 11x11. Essa variação de kernel foi essencial para observar os diferentes níveis de suavização:

    a) Kernel 3x3: Suavização leve, adequada para preservação de detalhes.

    b) Kernel 5x5, 7x7: Suavização moderada, com maior remoção de ruídos sem perda excessiva de nitidez.

    c) Kernel 11x11: Suavização intensa, com remoção significativa de ruído, porém com possível perda de detalhes.

    - Exibição e Salvamento dos Resultados:

    Após aplicar cada filtro, o resultado foi exibido na janela "Smoothing Demo" e salvo automaticamente com um nome específico que identificava o tipo de filtro e o tamanho do kernel, facilitando a comparação dos resultados posteriormente.

    Os nomes dos arquivos foram padronizados, como homogeneous_blur_3x3.jpg, gaussian_blur_5x5.jpg, entre outros, para organização dos dados gerados.
