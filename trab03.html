Trabalho - Parte III - Desenvolvimento do Sistema de Processamento Visual (SPV)

O Sistema de Processamento Visual (SPV) foi desenvolvido atendendo aos requisitos técnicos e pedagógicos estabelecidos, garantindo a entrega de um software funcional e alinhado às necessidades do projeto.

Linguagem e Biblioteca: O programa foi implementado em C++ utilizando a API OpenCV, que proporcionou um conjunto robusto de ferramentas para o processamento de imagens e visão computacional.

Plataforma de Execução: O sistema foi projetado e testado para execução no sistema operacional Ubuntu-Linux, utilizando a webcam disponível no laboratório didático para a captura de imagens e vídeos.

Desenvolvimento Exclusivo: O código foi integralmente desenvolvido pelos integrantes da equipe, respeitando a integridade acadêmica e sem recorrer a trabalhos pré-existentes de terceiros.

Bibliotecas Permitidas: Foram utilizadas bibliotecas de processamento visual reconhecidas e de uso público e gratuito, garantindo a aderência às especificações do projeto.

Processamento em Tempo Real: O programa realizou cálculos “on-the-fly” baseados na teoria e no modelo matemático adotado, permitindo a visualização contínua do vídeo ou das imagens processadas, garantindo respostas rápidas durante a execução.

Documentação no Código: Todos os arquivos de código incluíram um cabeçalho contendo:

  • Nome completo dos integrantes.
  • RA dos integrantes.
  • Data de criação do programa.
  • Nome do programa.
  • Exemplo de chamada do programa no terminal do Linux.

Personalização das Janelas: As janelas geradas pelo programa exibiram o título do software e o nome da equipe no cabeçalho, facilitando a identificação visual durante a execução.

Atendimento ao Contexto: O sistema desenvolvido atendeu a todas as demandas e necessidades previstas pelo Contexto e Cenário de Aplicação (CA), proporcionando uma solução funcional e acessível para usuários leigos, que puderam executar o programa e visualizar os resultados obtidos de forma clara e intuitiva.
