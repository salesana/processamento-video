Trabalho - Parte II - Modelagem Funcional do Sistema de Processamento Visual (MF)

O sistema de processamento visual utiliza técnicas avançadas de visão computacional para realizar a detecção e o reconhecimento facial. Ele é baseado em um fluxo modular que combina o algoritmo Haar Cascade para detecção de faces e o Local Binary Patterns Histogram (LBPH) para reconhecimento facial. Esses métodos foram escolhidos devido à sua eficiência e capacidade de operar em tempo real, mesmo em ambientes com restrições de recursos computacionais. As principais funcionalidades detalhadas do sistema são:

Captura de Imagens: Utilizamos o módulo implementado em takephotos.cpp para capturar imagens faciais de indivíduos através de uma câmera conectada ao dispositivo. As imagens são processadas e associadas a identificadores únicos armazenados no arquivo id-names.csv. As imagens capturadas são salvas no diretório faces, organizadas por identificadores.

Treinamento de Modelos: O sistema utiliza o diretório faces como fonte de dados para o treinamento do classificador LBPH. Durante esse processo, as imagens são convertidas para escala de cinza, e os histogramas de padrões binários locais são calculados para cada região da face. Esses histogramas são armazenados no diretório Classifiers como modelos treinados, que serão utilizados no reconhecimento facial.

Detecção Facial com Haar Cascade: Para identificar a localização de faces em imagens ou fluxos de vídeo, o sistema utiliza o classificador Haar Cascade. Esse método é baseado na aplicação de filtros Haar para detectar características faciais, como olhos, nariz e boca, dentro de janelas de diferentes tamanhos. Sua estrutura de cascata permite que a detecção seja rápida e eficaz, descartando rapidamente regiões que não contêm faces.

Reconhecimento Facial com LBPH: O reconhecimento facial é realizado utilizando o LBPH. Após a detecção da face, o sistema compara o histograma da face capturada com os histogramas armazenados nos modelos treinados. Essa abordagem é robusta a variações de iluminação e expressões faciais, garantindo alta precisão no reconhecimento.

Estrutura Modular: A arquitetura do sistema é projetada de forma modular para facilitar a manutenção e futuras expansões. Cada bloco funcional (captura, detecção, treinamento e reconhecimento) é independente, permitindo atualizações ou substituições sem afetar os outros componentes.

Fluxograma: O fluxo do sistema pode ser descrito da seguinte forma:

  • 01. Captura de imagens faciais e associação com identificadores únicos.
  • 02. Processamento das imagens capturadas para treinamento do modelo LBPH.
  • 03. Detecção facial em tempo real utilizando Haar Cascade.
  • 04. Reconhecimento facial comparando o histograma capturado com os modelos LBPH treinados.
  • 05. Exibição do identificador correspondente ou mensagem de "não encontrado".

Cenário de Implementação: O sistema é ideal para aplicações que requerem reconhecimento facial em tempo real, como controle de acesso, autenticação e vigilância. Ele é capaz de lidar com grandes volumes de dados, mantendo um desempenho eficiente. Alterações futuras podem incluir a integração com redes neurais profundas para aumentar a precisão do reconhecimento.
